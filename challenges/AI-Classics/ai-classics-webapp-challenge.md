## Overview
Develop a web application that showcases the top 10 most famous movies, influential ideas, or classic games (such as chess) related to Artificial Intelligence. The app should provide engaging content, visuals, and brief descriptions for each entry, highlighting their impact on AI history and culture.

## Challenge Goals
    - Frontend frameworks: React, Vue, Angular, Svelte, plain HTML/CSS/JS
    - Python (Flask, Django, FastAPI)
    - Java (Spring Boot, Jakarta EE)
    - C# (ASP.NET Core, Blazor)
    - Ruby (Rails)


## Summary
This challenge will help you explore AI's cultural impact through movies, ideas, and games, while building a polished web app. Document your process and share your learnings!

### What is Agent Mode?
Agent Mode (also called Copilot Edits) allows Copilot to make changes across multiple files autonomously. Instead of just suggesting code, it can actually edit files, create new files, and refactor code based on your instructions. Think of it as Copilot taking control of the keyboard!

### Lab 4: Prompt Files (5 Minutes)
**Objective:** Create reusable prompt files for consistent, repeatable workflows.

### What are Prompt Files?
Prompt Files (.prompt.md files) are reusable prompts you can invoke with a simple command. Instead of typing the same instructions over and over, you create a prompt file once and reuse it anytime! They're perfect for:
- Generating tests
- Code reviews
- Documentation
- Refactoring tasks
- Any repetitive workflow

**Exercise 4.1: Understanding Existing Prompt Files**
Check your project for any .prompt.md files or similar prompt files. Review their contents to understand how they guide Copilot for tasks like test generation, code reviews, or documentation. If none exist, consider creating one for a common workflow.

**Exercise 4.2: Create and Use Prompt Files**
**Exercise 4.3: Create a Documentation Prompt**
Create a new prompt file (e.g., documentation.prompt.md). Add instructions for documenting new features, components, or workflows. Include guidelines for structure, clarity, and AI-themed comments. Use this prompt file whenever you need to generate or update documentation to ensure consistency and completeness.
Create a new prompt file (e.g., generate-tests.prompt.md). Add instructions for generating tests, reviewing code, or documenting features. Reference these files when working with Copilot to ensure consistency and quality.
